modalité X : $m_J^{X}$ (lignes), Y : $m_K^{Y}$ (colonnes)
	quantitatif continu = classes modales
	$J$ Colonnes, $K$ lignes
	$n_{jk}$ : les effectif en position $(m_j^{X},m_k^{Y})$
	$n_{\bullet k}$ : effectif conjoints de chaque $m_k^{Y}$
	$n_{j\bullet}$ : les effectif conjoints de chaque $m_j^{X}$

Fréquence conditionnelle
	Profil ligne / profil colonne $$\frac{f_{j1}}{f_{j \bullet}}, \frac{f_{j2}}{f_{j \bullet}}, ..., \frac{f_{jK}}{f_{j \bullet}}$$
	$A \subset P$ 
	$B \subset P$
	$f_{A \mid P} = \frac{n_A}{n_P}$ : Fréquence de A dans P
	$f_{A \mid B} = \frac{n_{AB}}{n_B}$ : Ce qu'il reste de A dans B (A$ \land B$)
